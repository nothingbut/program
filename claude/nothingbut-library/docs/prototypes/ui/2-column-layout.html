第一章 科学边界

汪淼开车驶过北京的午夜街道，去参加一个奇怪的会议。这个会议通知是在晚上十一点钟发出的，通知人是警察。

不太可能是出了什么事，汪淼想，如果真是那样，警察会直接来找他的。会议通知说只占用他一个小时，就在附近的一个军事管理区内，他以前从未去过那里。

汪淼是一位纳米材料学家，从事的研究与军事和警察机构都没有什么联系。他是一个典型的知识分子，平时专心于工作，对政治和社会事务不感兴趣。
