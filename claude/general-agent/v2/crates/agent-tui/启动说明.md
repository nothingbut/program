## 🎮 快捷键说明

### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` / `Ctrl+Q` | 退出应用 |
| `Tab` | 切换焦点（会话列表 ↔ 输入框）|
| `ESC` | 取消焦点 |

### 会话列表（焦点在左侧时）

| 快捷键 | 功能 |
|--------|------|
| `j` / `↓` | 下一个会话 |
| `k` / `↑` | 上一个会话 |
| `Enter` | 选择会话并加载消息 |
| `n` | 新建会话 |
| `d` | 删除当前会话 |
| `r` / `F5` | 刷新会话列表 |

### 输入框（焦点在右下角时）

| 快捷键 | 功能 |
|--------|------|
| 输入文字 | 输入消息内容 |
| `Enter` | 发送消息 |
| `Backspace` | 删除字符 |
| `←` / `→` | 移动光标 |
| `Ctrl+U` | 清空输入 |

## 🎯 使用流程

1. **启动应用**: `./run_tui.sh`
2. **新建会话**: 按 `n` 创建新会话
3. **切换到输入框**: 按 `Tab`
4. **输入消息**: 直接输入文字
5. **发送消息**: 按 `Enter`
6. **查看响应**: 助手回复会逐段显示（流式响应）
7. **切换会话**: 按 `Tab` 回到会话列表，用 `j/k` 导航
8. **退出应用**: 按 `Ctrl+C`

## 🐛 常见问题

### 1. 启动时提示 "Ollama 未运行"

**解决方案**:
```bash
ollama serve
```

### 2. 提示 "未找到模型"

**解决方案**:
```bash
ollama pull qwen3.5:0.8b
```

### 3. 数据库文件权限问题

**解决方案**:
```bash
rm agent_tui_demo.db  # 删除旧数据库
./run_tui.sh           # 重新启动，会自动创建
```

### 4. 界面显示异常

**解决方案**:
- 确保终端大小至少 80x24
- 使用现代终端（iTerm2、Terminal.app、Alacritty 等）
- 检查终端颜色支持

### 5. 响应速度慢

**解决方案**:
- 使用更小的模型：`ollama pull qwen3.5:0.5b`
- 修改 `tui_demo.rs` 中的模型配置
- 检查网络连接（首次使用 Ollama）
