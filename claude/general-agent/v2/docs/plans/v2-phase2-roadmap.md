### Stage 5: RAG 集成 (3-4 天)
**状态:** ⏳ 待开始

**目标:**
- 文档检索增强
- 知识库管理
- 智能引用

**技术方案:**
- 向量存储: Qdrant（本地模式）
- Embedding: sentence-transformers（通过 Ollama）
- 检索: 混合检索（关键词 + 语义）

**任务清单:**
- [ ] agent-rag crate 创建
- [ ] 文档加载器（Markdown/PDF/Text）
- [ ] 文本分块策略
- [ ] Embedding 生成
- [ ] 向量存储集成
- [ ] 检索器实现
- [ ] 重排序逻辑
- [ ] 引用生成
- [ ] 与 ConversationFlow 集成
- [ ] CLI 命令（索引管理）
- [ ] 测试和文档

**功能特性:**
- 多格式文档支持
- 智能分块
- 混合检索
- 引用溯源
- 增量更新

**验收标准:**
- 检索准确率 > 80%
- 支持常见文档格式
- 引用可追溯
- 性能可接受（<2s）
