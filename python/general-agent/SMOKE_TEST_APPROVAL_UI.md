## 常见问题

### Q1: 运行报错 "ModuleNotFoundError"

**解决:**
```bash
# 确保在项目根目录
cd /Users/shichang/Workspace/program/python/general-agent

# 检查依赖
pip list | grep rich
```

### Q2: 界面显示乱码

**解决:**
```bash
# 确保终端支持 UTF-8
export LANG=en_US.UTF-8

# 或者使用 iTerm2/Windows Terminal
```

### Q3: 输入无响应

**解决:**
- 确保终端处于交互模式
- 检查是否有后台进程占用输入
- 尝试 Ctrl+C 重新启动

### Q4: 测试被标记为失败

**原因:** Smoke 测试根据你的选择判断成功/失败
- 测试 1: 期望批准 (Y)
- 测试 2: 期望拒绝 (N)
- 测试 3: 期望批准 (Y)
- 测试 4: 期望批准 (Y)

如果选择不符，测试会标记为失败（但这不代表代码有问题）。
